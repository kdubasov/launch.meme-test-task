TITLE: Setting Up Environment Variables - .env
DESCRIPTION: This snippet outlines the required environment variables for the project, including Cloudflare R2 storage credentials, a Solana RPC URL, and a pool configuration key. These variables must be set in a `.env` file in the root directory for the application to function correctly.

SOURCE: https://github.com/meteoraag/meteora-scaffold/blob/main/templates/fun-launch/README.md#_snippet_2

LANGUAGE: env
CODE:
```
# Cloudflare R2 Storage
R2_ACCESS_KEY_ID=your_r2_access_key_id
R2_SECRET_ACCESS_KEY=your_r2_secret_access_key
R2_ACCOUNT_ID=your_r2_account_id
R2_BUCKET=your_r2_bucket_name

# Solana RPC URL
RPC_URL=your_rpc_url

# Pool Configuration
POOL_CONFIG_KEY=your_pool_config_key
```
